# Student-Grade-Calculator

## Overview

This is a simple Python program that calculates and displays grades for students based on their marks. The program stores student marks in a list, assigns a grade according to predefined grading criteria, and prints the results.

## Features

* Stores marks for multiple students
* Automatically assigns grades based on marks
* Displays student number, marks, and grade
* Uses loops and conditional statements

## Technologies Used

* Python

## Grading Criteria

| Marks Range | Grade |
| ----------- | ----- |
| 91 - 99     | A     |
| 81 - 89     | B     |
| 71 - 79     | C     |
| 61 - 69     | D     |
| 51 - 59     | E     |
| Below 50    | F     |

## How It Works

1. Student marks are stored in a list.
2. The program checks each mark using conditional statements.
3. A corresponding grade is assigned.
4. The results are displayed for each student.

## How to Run

1. Ensure Python is installed on your system.
2. Save the file as `student_grades.py`.
3. Open a terminal or command prompt.
4. Run the following command:

```bash
python student_grades.py
```
